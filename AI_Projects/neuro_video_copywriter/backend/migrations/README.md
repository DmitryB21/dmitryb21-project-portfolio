# Миграции БД (Alembic)

- `env.py` — конфигурация Alembic, интегрированная с `backend.database.base`.
- `script.py.mako` — шаблон autogenerated-миграций.

## Как работать

```bash
poetry run alembic revision --autogenerate -m "init schema"
poetry run alembic upgrade head
```

Перед запуском убедитесь, что:

1. `NVC_DATABASE_URL` задан и указывает на нужную базу.
2. Все модели импортируются в `backend/database/models.py`.
3. Коллекция Qdrant создаётся отдельно (автогенерация не покрывает векторное хранилище).
