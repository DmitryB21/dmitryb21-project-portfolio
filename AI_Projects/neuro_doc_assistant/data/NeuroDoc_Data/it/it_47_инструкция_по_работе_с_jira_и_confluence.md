# Инструкция по работе с инструментами Atlassian — Jira и Confluence

## Введение
Jira и Confluence являются популярными решениями компании Atlassian для управления проектами и совместной работы команд. Данная инструкция предназначена для сотрудников, работающих с этими системами ежедневно.

## Шаг 1. Регистрация и авторизация в системах

### 1. Открытие браузера и переход на сайт системы
- Запустите браузер (Google Chrome, Mozilla Firefox, Safari).
- Перейдите на официальный сайт Atlassian:
  ```
  https://jira.atlassian.com/ (для Jira)
  https://yourcompany.confluence.atlassian.net/ (для Confluence)
  ```

### 2. Авторизация в системе
- Нажмите кнопку **«Авторизоваться»** или **«Login»**.
- Введите ваши учетные данные (логин и пароль), выданные отделом информационной безопасности вашей организации.

## Шаг 2. Работа с системой Jira

### 2.1 Создание нового проекта
- Перейдите в раздел **«Управление проектами»** → **«Создать проект»**.
- Выберите шаблон проекта (например, Agile или Kanban).
- Укажите название проекта, описание и выберите команду разработчиков.

### 2.2 Управление задачами и багами
- Войдите в интерфейс вашего проекта.
- Для добавления новой задачи нажмите **«Добавить новую задачу»**.
- Заполните поля:
  - Название задачи,
  - Описание проблемы или задачи,
  - Назначьте ответственное лицо (assignee),
  - Установите приоритет задачи.
  
### 2.3 Просмотр статуса задач и отчетность
- Используйте фильтры и отчеты для отслеживания прогресса команды.
- Перейдите в раздел **«Журнал задач»**, чтобы увидеть историю изменений задач.

## Шаг 3. Использование системы Confluence

### 3.1 Создание страницы документации
- Перейдите в раздел **«Пространства»** → **«Создать пространство»**.
- Задайте имя пространства и укажите цель создания (например, документация по продукту или инструкции).

### 3.2 Добавление содержимого страниц
- Создайте новые страницы и добавьте контент:
  - Заголовки,
  - Текстовые блоки,
  - Таблицы,
  - Формулы (используйте LaTeX для сложных формул).

### 3.3 Обсуждение и комментарии
- Активируйте обсуждения на страницах.
- Оставляйте комментарии и задавайте вопросы другим участникам команды.

## Заключение
После выполнения инструкций вы сможете эффективно использовать инструменты Jira и Confluence для повышения эффективности взаимодействия внутри команды и улучшения качества проектной деятельности.
