# Инструкция по работе с корпоративной электронной почтой

## 1. Общие положения
Корпоративная электронная почта предназначена исключительно для служебного общения сотрудников компании. Использование почты вне рабочих целей запрещено.

## 2. Вход в почтовый клиент
### Шаг 1. Запустите почтовое приложение (например, Microsoft Outlook).
- Откройте стартовое меню операционной системы.
- Найдите иконку почтового клиента и дважды кликните её.

### Шаг 2. Авторизация в почтовом приложении.
- Введите данные авторизации:
  - Логин — имя сотрудника в формате `<имя_фамилия@company.ru>`.
  - Пароль — пароль от личного аккаунта компании.
  
### Шаг 3. Подтверждение безопасности.
- Если требуется подтверждение безопасности учетной записи, следуйте инструкциям приложения.

## 3. Работа с входящей корреспонденцией
### Шаг 4. Просмотр входящих писем.
- Перейдите во вкладку «Входящие».
- Используйте фильтры и поисковые запросы для быстрого нахождения нужного письма.

### Шаг 5. Отметка прочитанных сообщений.
- Чтобы отметить письмо как прочитанное, щелкните правой кнопкой мыши на письме и выберите пункт «Прочитано».

### Шаг 6. Сохранение важных писем.
- Для сохранения письма нажмите клавишу `Ctrl + S` или используйте контекстное меню («Сохранить копию»).

## 4. Отправка писем
### Шаг 7. Создание нового письма.
- Нажмите кнопку «Создать новое сообщение» или комбинацию клавиш `Ctrl + N`.
- Укажите адрес получателя (не менее одного), тема письма должна быть четкой и понятной.

### Шаг 8. Написание текста письма.
- Используйте стандартные правила деловой переписки: приветствие, основной текст, подпись.
- Не отправляйте конфиденциальные сведения через открытую почту.

### Шаг 9. Прикрепление файлов.
- Выберите нужные файлы и перетащите их в окно отправки письма.
- Проверьте корректность прикрепленных документов перед отправкой.

### Шаг 10. Отправка письма.
- После завершения ввода текста и проверки вложений нажмите кнопку «Отправить».

## 5. Ответ на входящие письма
### Шаг 11. Получение и обработка запроса.
- Прочтите внимательно письмо, уточните детали, если нужно.
- Составьте ответ согласно требованиям делового этикета.

### Шаг 12. Формирование ответа.
- Создайте новое письмо и укажите исходный документ в качестве цитаты (`Shift + Enter`).
- Добавьте ссылку на архивированный файл оригинала письма, если требуется.

### Шаг 13. Завершение обработки письма.
- Отправьте ответ сотруднику, убедившись, что вложенные файлы правильно прикреплены.

## 6. Настройки профиля и безопасности
### Шаг 14. Изменение настроек профиля.
- Войдите в настройки почтового клиента.
- Установите актуальные контактные данные, фотографию и прочие параметры.

### Шаг 15. Установка антивирусной защиты.
- Включите автоматическое сканирование всех входящих и исходящих писем.
- Регулярно обновляйте антивирусные базы.

## Заключение
Соблюдение данной инструкции позволяет эффективно использовать корпоративную электронную почту, обеспечивая безопасность, качество и скорость обмена информацией внутри организации.
