# Инструкция по сбросу пароля учётной записи в Active Directory

## Введение
Данная инструкция предназначена для администратора информационной системы организации и описывает порядок действий по восстановлению утраченного пароля пользователя в домене Microsoft Active Directory (AD).

## Шаги по сбросу пароля

### 1. Подключение к консоли управления Active Directory
- Откройте оснастку **Active Directory Users and Computers** через меню «Пуск» → «Администрирование» → «Active Directory пользователей и компьютеров».
  
### 2. Выбор нужного объекта
- Перейдите в раздел с пользователями, найдите запись пользователя, пароль которого нужно сбросить.
- Щёлкните правой кнопкой мыши на выбранном пользователе и выберите пункт **Свойства**.

### 3. Сброс пароля через свойства пользователя
- Во вкладке **«Общие»** нажмите кнопку **«Сбросить пароль»**, подтвердив действие нажатием кнопки **«Да»**.
- Введите новый пароль дважды в появившихся полях ввода.
- Нажмите кнопку **«ОК»** для завершения операции.

### 4. Проверка успешного изменения пароля
- После выполнения указанных шагов рекомендуется проверить корректность введённого пароля, зайдя от имени пользователя в систему.

### Альтернативный способ — использование командлета PowerShell
- Запустите окно PowerShell с правами администратора.
- Выполните команду:

```powershell
Set-ADAccountPassword <имя_учётной_записи> -NewPassword (ConvertTo-SecureString "<новый_пароль>" -AsPlainText -Force)
```

- Пример команды:

```powershell
Set-ADAccountPassword IvanovIP -NewPassword (ConvertTo-SecureString "Qwerty123!" -AsPlainText -Force)
```

- Команда автоматически обновит пароль указанного пользователя.

### Дополнительная проверка после сброса пароля
- Рекомендуется убедиться, что учётная запись успешно авторизуется и доступ к ресурсам восстановлен корректно.

## Важные замечания
- Не сохраняйте пароль пользователя открытым текстом ни в каком виде.
- Пароль должен соответствовать политике безопасности компании и требованиям сложности паролей AD.
- Если учётная запись заблокирована, перед изменением пароля убедитесь, что блокировка снята командой:

```powershell
Unlock-ADAccount <имя_учётной_записи>
```

## Заключение
После выполнения данной инструкции администратор должен удостовериться, что учётная запись активна и готова к использованию.
